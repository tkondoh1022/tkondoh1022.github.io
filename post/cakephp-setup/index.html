If you already have sample project, delete it.

1$ sudo rm -rf ./data/htdocs/sample

Create sample project.

1$ docker-compose exec phpfpm /bin/sh
2/var/www/html # curl -s https://getcomposer.org/installer | php

Add phpmyadmin

Add following lines in docker-compose.yml.

 1  phpmyadmin:
 2    image: phpmyadmin/phpmyadmin:latest
 3    ports:
 4      - 8888:80
 5    environment:
 6      - PMA_ARBITRARY=1
 7      - PMA_HOST=mysql
 8      - PMA_USER=root
 9      - PMA_PASSWORD=password
10    depends_on:
11      - mysql
12      - phpfpm
13      - nginx
14    # Connect a container to another container
15    links:
16      - mysql
17      - phpfpm
18      - nginx
