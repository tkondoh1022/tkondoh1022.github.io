I made my blog site with Hugo and easy-hugo.

  • https://www.ted027.com/post/hugo/

SPEC.

  • url: https://tkondoh1022.github.io
  • os: ubuntu18.04
  • spacemace and easy-hugo

SETUP

Install latest Hugo

1$ sudo snap install hugo --channel=extended
2$ sudo ln -s /sbin/hugo /usr/bin/hugo

Setup GitHub

Make two repos

  • blog
  • tkondoh1022.github.io

Set new site and first post

 1$ cd ~/work-github
 2$ hugo new site blog
 3$ cd blog
 4$ cd theme
 5$ git clone https://github.com/mattbutton/silhouette-hugo.git
 6$ rm -rf silhouette-hugo/.git
 7$ cd ../
 8
 9$ cat config.toml
10---
11baseurl = "https://tkondoh1022.github.io"
12theme = "silhouette-hugo"
13defaultContentLanguage = "ja"
14---
15
16$ hugo new post/setup-blog.md
17$ edit content/post/setup-blog.md

Push repos.

 1$ git init
 2$ git remote add origin git@github.com:tkondoh1022/blog.git
 3$ emacs .gitignore
 4$ git add -A
 5$ git commit -m "initial commit"
 6$ git push origin master
 7$ git merge --allow-unrelated-histories origin/master
 8$ git submodule add git@github.com:tkondoh1022/tkondoh1022.github.io.git public
 9
10$ hugo
11$ cd public
12$ git add -A
13
14$ git commit -m "initial article"
15$ git push origin master

EASY-HUGO

Setup .spacemacs

1--- .spacemace ---
2  ;;************************************************
3  ;;; easy-hugo
4  ;;************************************************
5  (setq easy-hugo-basedir "~/work-github/blog/")
6  (setq easy-hugo-url "https://tkondoh1022.github.io")
7  (setq easy-hugo-postdir "content/post")
8  (define-key global-map (kbd "C-c C-e") 'easy-hugo)
9  )
